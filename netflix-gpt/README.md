# STEPS TO DEPLOYMENT:

1. Install firebase CLI - `npm i -g firebase-tools`
2. Firebase login - `firebase login`
3. Initialized Firebase - `firebase init`, then select Hosting (optionally)
4. Deploy command - `firebase deploy`
